## Usage

```jsx
import { RicosEditor } from 'ricos-editor';
import { pluginGiphy } from 'wix-rich-content-plugin-giphy';
import 'ricos-editor/dist/styles.min.css';
import 'wix-rich-content-plugin-giphy/dist/styles.min.css';

/* Editor */
<RicosEditor plugins={[pluginGiphy(GiphyConfig)]} />

/* Viewer */
<RicosViewer plugins={[pluginGiphy(GiphyConfig)]} />
```

## Giphy Config API

### `giphySdkApiKey`

```jsx
giphySdkApiKey: string;
```

Get your Giphy API Key <a href="https://support.giphy.com/hc/en-us/articles/360020283431-Request-A-GIPHY-API-Key">here</a>

### `sizes`

Control GIF size

```jsx
sizes?: {
  desktop?: 'original' | 'downSizedSmall',
  mobile?: 'original' | 'downSizedSmall'
}
```

Default value:

```jsx
{
  desktop: 'original',
  mobile: 'original'
}
```
