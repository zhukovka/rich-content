Introducing the rich content viewer

Now that you have an editor, let's add a viewer to display the content it creates.

npm i ricos-viewer
import { RicosViewer } from 'ricos-viewer';
import 'ricos-viewer/dist/styles.min.css';
const content = {

import { pluginVideo } from 'wix-rich-content-plugin-video/dist/module.viewer.cjs';
import 'wix-rich-content-plugin-video/dist/styles.min.css';
import { pluginDivider } from 'wix-rich-content-plugin-divider/dist/module.viewer.cjs';
import 'wix-rich-content-plugin-divider/dist/styles.min.css';
