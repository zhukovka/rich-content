Quick Start

Start from installing the packages

npm i ricos-editor // TODO: complete the required package list

Then add a basic rich text editor to your app. Try typing, and add some formatting to your text.

import { RicosEditor } from "ricos-editor";
<RicosEditor placeholder={'Type here!'} />

// TODO: update when RicosEditor is merged

Type here!

Now let's add some plugins.

npm i wix-rich-content-plugin-video wix-rich-content-plugin-divider

This is how you can add videos and dividers.

import { RicosEditor } from "ricos-editor";
import { pluginVideo } from "wix-rich-content-plugin-video";
import "wix-rich-content-plugin-video/dist/styles.min.css";
import { pluginDivider } from "wix-rich-content-plugin-divider";
import "wix-rich-content-plugin-divider/dist/styles.min.css";
<RicosEditor plugins={[pluginDivider(), pluginVideo()]} placeholder={'Type here!'} />
Type here!

There you go! A rich content editor with plugins. Now go ahead and add some more plugins!
